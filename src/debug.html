Diagnostic & Debugger

Controls

captureLogs false

Captures warning() and log() calls from the Service Worker. Errors are always captured regardless of this flag. Persists across Service Worker restarts. Enable before reproducing a bug, then download the report.

discardInPlaceOfSuspend false

Tabs are discarded (Chrome unloads them from memory without navigating away) instead of being redirected to a suspended page. Useful for testing native discard behaviour. Not recommended for normal use.

claim suspended tabs run claim

Re-assigns suspended tabs that point to a different extension ID to this instance. Use after reinstalling or reloading the unpacked extension if suspended tabs show a broken page.

news feed force refresh

Forces an immediate RSS fetch, bypassing the 24 h TTL cache. Use to verify that newly published posts are picked up without waiting for the next scheduled refresh.
